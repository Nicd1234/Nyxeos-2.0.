# Nyxeos 2.0

Application complète de gestion et organisation des tâches quotidiennes.

## Fonctionnalités
- Ajout, édition, suppression et complétion de tâches
- Priorités, catégories, échéances
- (À venir : Pomodoro, habitudes, stats, Kanban...)

## Lancement rapide

1. Backend :
   cd backend
   npm install
   npx prisma migrate dev --name init
   npm run dev

2. Frontend :
   cd frontend
   npm install
   npm run dev

Ouvrez http://localhost:5173
